# SOFIPOS-2024
Un tema muy común que me preguntan actualmente es ¿dónde invertir?, y la verdad es que existen una gran variedad de instrumentos de inversión, no solo acciones y CETES, en México existen las SOFIPOS y como buen financiero toca no solo ver el rendimiento de mi inversión, sino igual entender su funcionamiento y un análisis exhaustivo del funcionamiento de estas entidades. 

Así que decidí descargar los reportes de 2024 que publica la CNBV, limpiarlas primero en excel, luego utilizar Power Query para mejorar la limpieza y finalmente estructurar estos datos en Dashboard en Power BI para visualizar desde las tasas hasta el crecimiento de estas mismas entidades.

Finalmente recopile toda esta información en un reporte en pdf que puedes leer con mucho gusto.
